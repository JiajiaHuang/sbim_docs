## 使用提示

:::tip 最佳实践
- 在执行重要任务前，建议先进行测试飞行
- 定期检查设备状态和维护记录
- 保持软件版本更新以获得最新功能
:::

:::warning 安全提醒
- 始终遵守当地无人机飞行法规
- 确保飞行区域安全，避开人群和障碍物
- 在恶劣天气条件下避免飞行
:::
